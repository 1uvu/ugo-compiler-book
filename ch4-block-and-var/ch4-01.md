# 4.1 赋值和词法域例子

赋值涉及变量的定义和赋值，词法域则对应嵌套的大括弧。本章我们尝试支持以下代码的编译：

```go
package main

func main() {
	var x int
	var y int = 123

	x = 1 + 2 + 3
	println(x+100)

	x = x + 10
	println(x+100)

	{
		y := y + 1
		println(y)
	}
	println(y)

	var z = 456
	println(z)
}
```

main函数首先第一x、y、z三个局部变量，然后以不同的形式初始化。然后通过修改x变量的值并输出查看其状态变化。其中嵌套的大括弧在内层词法域定义了一个同名的y变量，修改内部y后分别输出内部y和外部同名的y查看其状态。
